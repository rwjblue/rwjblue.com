# rwjblue.com

Personal landing page for `rwjblue.com`, also served at `n1rwj.com` and
`n1rwj.radio`.

## Local Development

This project uses [mise](https://mise.jdx.dev/) for tool versions and file-based
tasks.

```bash
mise install
mise run install
mise run dev
```

## Validation

```bash
mise run check
mise run build
```

## Deployment

The site is designed for Cloudflare Pages. See
