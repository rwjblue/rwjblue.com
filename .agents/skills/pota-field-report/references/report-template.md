operation, but keep the field report easy to scan.

## Style Notes

The best reports read like practical portable-radio field notes rather than a
generic trip report:

- Anchor the note in a real operating decision or constraint.
- Let the prose explain setup choices and field tradeoffs.
- Include enough site texture to remember the activation later.
- Report the on-air result plainly: bands, modes, QSO pace, and stopping point.
- Keep paragraphs short enough to read on a phone.
- End with repeatable lessons when there are any.
- Mention QRP only when the user's notes identify the operation as QRP.
